# カラタチの最果てのセンセイ！ 切り抜きサンプラー

ポッドキャスト「カラタチの最果てのセンセイ！」の切り抜きを、8個のPADでランダムに楽しむWebアプリです。

## 現在の収録内容

- 第1回〜第9回
- 切り抜き76本
- MP3は `public/audio/` に格納
- MP3のID3タイトルから `data/clips.json` を生成

## 主な機能

- ページを開くたびにランダムな8個を表示
- PADを押すと即時再生
- お気に入りをブラウザに保存
- お気に入り済みPADにハートを表示
- 今週／累計の人気ランキング
- Apple Podcast本編へのリンク
- PC・スマートフォン対応

## 音声を追加する場合

1. `ep010-01.mp3` のような名前で `public/audio/` に追加します。
2. MP3の「タイトル」タグに切り抜きタイトルを入れます。
3. `npm run sync:clips` を実行して `data/clips.json` を更新します。

長期運用ではGitHub Pagesの容量制限を避けるため、MP3をオブジェクトストレージへ移し、`audioUrl` から参照する構成を予定しています。

## 開発

```bash
npm install
npm run dev
```

現在のソースはChatGPT Sites向けのサーバー構成を含みます。GitHub Pagesへ公開する際は、ランキングAPIを外部バックエンドへ接続し、静的サイト用のビルド設定を追加する必要があります。
